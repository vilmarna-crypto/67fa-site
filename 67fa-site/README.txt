# 67FA — Netlify Drop-in Site

This folder is ready for Netlify's drag‑and‑drop deploy.

## How to deploy
1) Download and unzip this folder.
2) On https://app.netlify.com/ — click **Add new site → Deploy manually**.
3) Drag the **entire folder** (not just a file) onto the drop zone. You can also drag the ZIP.
4) When it finishes, click the site URL to view.

Notes:
- All code is in **index.html**. React and Tailwind are loaded via CDN. No build step needed.
- To edit text, open index.html in any editor and save.
- Replace the registration link with your Google Form / GotSport URL.
- To use your domain (67FA.com), add it in Site settings → Domain management.
